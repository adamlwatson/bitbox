ArduinoUnit Documentation

ArduinoUnit Library Reference

test()

Description

Defines a test function in the most recently declared test suite.

Syntax

test(<name>)

Parameters

<name>: the name of the test function

Example

#include <ArduinoUnit.h> TestSuite suite; void setup() { } test(addition) { assertEquals(2, 1 + 1); } void loop() { suite.run(); }
