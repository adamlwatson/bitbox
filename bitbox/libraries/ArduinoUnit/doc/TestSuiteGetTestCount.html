ArduinoUnit Library Reference

getTestCount()

Description

Returns the number of tests within a test suite.

Syntax

suite.getTestCount()

Parameters

suite: the test suite to query

Returns

the number of tests within the suite as an integer

Example

#include <ArduinoUnit.h> TestSuite suite; void setup() { } test(first) { assertTrue(true); } test(second) { assertTrue(true); } void loop() { // testCount == 2 int testCount = suite.getTestCount(); }
