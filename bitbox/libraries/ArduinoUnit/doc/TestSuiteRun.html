ArduinoUnit Library Reference

run()

Description

Runs a test suite.

Syntax

suite.run()

Parameters

suite: the test suite to run

Returns

true if the test suite ran successfully, false if a fatal error (such as a failure to allocate memory) occurred. Note that true will be returned even if test failures occur.

Example

#include <ArduinoUnit.h> TestSuite suite; void setup() { } void loop() { suite.run(); }
