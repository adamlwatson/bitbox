ArduinoUnit Library Reference

reportEqualityFailure()

Description

Reports a test failure from an assertEquals() assertion.

Syntax

reportEqualityFailure(test, lineNumber, expected, actual)

Parameters

test: the failing test
lineNumber: the line number on which the test failed
expected: the expected value (as a string)
actual: the actual value (as a string)
