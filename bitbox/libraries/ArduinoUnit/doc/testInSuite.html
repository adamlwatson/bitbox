ArduinoUnit Library Reference

testInSuite()

Description

Defines a test function in the specified test suite.

Syntax

testInSuite(<name>)

Parameters

<name>: the name of the test function

Example

#include <ArduinoUnit.h> TestSuite suiteOne; void setup() { } test(inSuiteOne) { assertTrue(true); } TestSuite suiteTwo; test(inSuiteTwo) { assertTrue(true); } testInSuite(inSuiteOneAgain) { assertTrue(true); } void loop() { // Run the two tests in suiteOne suiteOne.run(); // Run the single test in suiteTwo suiteTwo.run(); }
