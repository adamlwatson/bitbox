ArduinoUnit Library Reference

getFailureCount()

Description

Returns the number of failed tests within a test suite.

Syntax

suite.getFailureCount()

Parameters

suite: the test suite to query

Returns

the number of failed tests within the suite as an integer

Example

#include <ArduinoUnit.h> TestSuite suite; void setup() { } test(first) { assertTrue(true); } test(second) { assertTrue(false); } void loop() { suite.run(); // failureCount == 1 int failureCount = suite.getFailureCount(); }
